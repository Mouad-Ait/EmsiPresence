# EmsiPresence
. Développement d’une application Android de gestion des absences des étudiants
•	Création d’une page de démarrage (Splash Screen)
•	Conception des pages de connexion et d’enregistrement (Firebase Auth)
•	Localisation des différents sites de l’école
•	Intégration d’un assistant virtuel (Gemini)
•	Stack utilisée: Java, Android Studio, Firebase
